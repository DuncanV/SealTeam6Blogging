TO RUN
- Download and Run Docker Desktop - make sure Docker Desktop is working
- Navigate inside the "docker" folder
- double click the "build-docker.bat"
- let it install and compile (takes about 2 mins or so)

You can now visit http://localhost:3000


Please dont break it :) 

About the passwords
- Needs to be 10 chars
- Must contain lowercase
- Must contain uppercase
- Must contain numbers
- Must contain special characters
